## 开发流程

### 开发环境要求

-   Node.js >= 12 

### 开发

```sh
npm i
npm run dev
```

### 编译

```sh
npm run build
```

### 风格检查

```sh
npm run lint
```

### 文档开发

```sh
npm run docs:dev
```

### 文档编译

```sh
npm run docs:build
```

### 提交变更

```sh
npm run commit #请勿直接提交git commit
```
